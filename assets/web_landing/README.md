# Web Landing Assets

Place your assets here:

## Screenshots
- `screenshots/` - App screenshots for carousel
  - `screenshot_1.png` - Dashboard screenshot
  - `screenshot_2.png` - Seat grid screenshot
  - `screenshot_3.png` - Analytics screenshot
  - `screenshot_4.png` - Payment screenshot

## Logos
- `logos/` - Branding assets
  - `app_logo.png` - Main app logo (transparent background recommended)
  - `app_logo_white.png` - White version for dark backgrounds

## Notes
- Recommended screenshot size: 1080x1920 (portrait) or 1920x1080 (landscape)
- Logo should be at least 512x512px with transparent background
- Use PNG format for best quality
